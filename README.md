# PodoVital Alto Hospicio - Version 06

## Fecha

06 de junio de 2026

## Objetivo de la version

Actualizar la identidad comercial del sitio a PodoVital e incorporar el Instagram oficial entregado por el cliente.

## Cambios realizados

- Se creo `Version-06` sin sobrescribir `Version-05`.
- Se mantuvo una sola carpeta `assets`, sin subcarpetas internas.
- Se cambio el nombre visible de la pagina a `PodoVital`.
- Se actualizo el titulo SEO, Open Graph y datos estructurados con `PodoVital Alto Hospicio`.
- Se reemplazo el placeholder de Instagram por `https://www.instagram.com/podovital_alto_hospicio/`.
- Se actualizo la cabecera, footer, copyright y etiqueta accesible del logo.
- Se mantuvieron WhatsApp real, direccion real, mapa actualizado y seccion de casos clinicos.

## Mejoras implementadas

- Marca mas clara y consistente para presentacion a cliente real.
- Instagram oficial conectado desde la cabecera y la seccion de contacto.
- Mejor coherencia SEO local entre nombre comercial, servicio y ubicacion.

## Observaciones para futuras versiones

- Reemplazar `https://tudominio.cl/` en canonical, Open Graph, sitemap y JSON-LD cuando exista el dominio final.
- Mantener todos los assets directamente dentro de `assets`.
- Si se agregan mas casos o nuevos datos comerciales, crear `Version-07` y no sobrescribir esta version.
