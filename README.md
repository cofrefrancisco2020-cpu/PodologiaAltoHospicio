# Podologia Alto Hospicio - Version 05

## Fecha

05 de junio de 2026

## Objetivo de la version

Refinar la seccion de casos clinicos para que las leyendas de antes, despues y sesiones queden integradas sobre cada imagen, manteniendo una lectura mas ordenada, compacta y profesional.

## Cambios realizados

- Se creo `Version-05` sin sobrescribir `Version-04`.
- Se mantuvo una sola carpeta `assets`, sin subcarpetas internas.
- Se conservaron las imagenes aprobadas y optimizadas de `Version-04`.
- Se transformaron las leyendas inferiores de los casos en etiquetas flotantes sobre cada imagen.
- Se reemplazaron textos largos de panel por etiquetas compactas como `Antes` y `Despues`.
- Se ajustaron las posiciones para casos de 2 etapas, 3 etapas y 4 sesiones.
- Se mantuvieron WhatsApp real, direccion real y mapa actualizado.

## Mejoras implementadas

- Seccion de casos mas compacta y con mejor proporcion visual.
- Tarjetas de tratamientos con alturas mas consistentes.
- Lectura mas clara del antes/despues sin ocupar espacio extra bajo la imagen.
- Mejor comportamiento responsive de las etiquetas en celulares.

## Observaciones para futuras versiones

- Reemplazar `INSTAGRAM_AQUI` por el usuario real de Instagram.
- Reemplazar `https://tudominio.cl/` en canonical, Open Graph, sitemap y JSON-LD cuando exista el dominio final.
- Mantener todos los assets directamente dentro de `assets`.
- Si se agregan mas casos o nuevos datos comerciales, crear `Version-06` y no sobrescribir esta version.
